# LCD

A Elm powered solution for the ["Number to LCD" kata](https://codingdojo.org/kata/NumberToLCD/).

This program formats a `List Int` into `String` using the "LCD font" with desirable size.

## Examples

- Font 1x1
    ```text
       _  _     _  _  _  _  _
     | _| _||_||_ |_   ||_||_|
     ||_  _|  | _||_|  ||_| _|
    ```

- Font 5x2
    ```text
            _____  _____  _____
          |      |      |      |
          | _____| _____|      |
          |      |      |      |
          | _____| _____|      |
    ```
- Font 1x3
    ```text
     _     _     _     _
    | |  || |  || |  || |  |
    | |  || |  || |  || |  |
    | |  ||_|  || |  ||_|  |
    | |  || |  || |  || |  |
    | |  || |  || |  || |  |
    |_|  ||_|  ||_|  ||_|  |
    ```
